    Import Marks from CSV / Excel
    
        
    Upload a file — columns are auto-mapped to subjects, out-of-range values are flagged before import
    
      
    
      
    
    
    
    

      
      
    
        
    
          
          
    Drop your file here, or click to browse
    
          
    Supports CSV (.csv) and Excel (.xlsx / .xls)
    
          
        
    

        
    
          
    Expected file format
    
          
    
            Your file should have one row per student. Columns can include Student Name (or Index No.), then one column per subject.
            Column headers are matched to your subject names automatically — even fuzzy matches work!
    
            Partial uploads are supported: any subjects not in the file can be entered manually.

    Managing Subjects
    
        
    

          
    
            
    
              
              
            
    
            
    
              
    Move a subject to another group
    
              
    Hold and drag the ⠿ grip icon on any subject tag, then drop it onto a different group row. The subject moves instantly and a confirmation toast appears.
    
            
    
          
    

          
    
            
    
              
              
            
    
            
    
              
    Reorder subjects within a group
    
              
    Drag a subject left or right using the same grip icon to change its position inside its group. A blue caret line shows exactly where it will land — left half = before, right half = after.
    
            
    
          
    

          
    
            
    
              
              
            
    
            
    
              
    Edit a subject name or max marks
    
              
    Tap the pencil ✏️ on any subject tag to open the inline editor. You can tap freely between the name field and the marks field — the editor stays open until you tap ✓ to save or ✕ to cancel.
    
            
    
          
    

          
    
            
    
              
              
            
    
            
    
              
    Tip: hold for a moment before dragging
    
              
    On touch screens, press the grip dot slightly longer (~200 ms) before you start sliding — this prevents accidental drags while scrolling the page.
