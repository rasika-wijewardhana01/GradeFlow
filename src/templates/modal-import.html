Supports CSV (.csv) and Excel (.xlsx / .xls)
Expected file format
Your file should have one row per student. Columns can include Student Name (or Index No.), then one column per subject. Column headers are matched to your subject names automatically — even fuzzy matches work!
Partial uploads are supported: any subjects not in the file can be entered manually.
